To build pjbartlein.github.io webpage

1. Edit source files:

/Users/bartlein/Dropbox/WebPages/source_pjbartlein.github.io/source/BartleinAcadBio.docx
/Users/bartlein/Dropbox/WebPages/source_pjbartlein.github.io/source/BartleinCV-May2024.docx
/Users/bartlein/Dropbox/WebPages/source_pjbartlein.github.io/source/BartleinPublications.docx

2. export as .pdfs

3. run Pandoc to make .md files from .docx files

pandoc -s BartleinCV-Oct2025.docx -t markdown -o BartleinCV.md 
pandoc -s BartleinAcadBio.docx -t markdown -o BartleinAcadBio.md 
pandoc -s BartleinPublications.docx -t markdown -o BartleinPublications.md 

4. use pandoc to create .html files (pdf-wide.css)

pandoc -s BartleinCV.md -o BartleinCV.html --css pdf-wide.css
pandoc -s BartleinAcadBio.md -o BartleinAcadBio.html --css pdf-wide.css
pandoc -s BartleinPublications.md -o BartleinPublications.html --css pdf-wide.css

5. Build R Markdown .html web site in the folder /pjbartlein.github.io

(/Users/bartlein/Dropbox/WebPages/Bartlein/pjbartlein.github.io/Bartlein.Rproj)

6. Copy .html and .pdf files to /pjbartlein.github.io

/Users/bartlein/Dropbox/WebPages/Bartlein/pjbartlein.github.io/source/BartleinAcadBio.html
/Users/bartlein/Dropbox/WebPages/Bartlein/pjbartlein.github.io/source/BartleinCV.html
/Users/bartlein/Dropbox/WebPages/Bartlein/pjbartlein.github.io/source/BartleinPublications.html
/Users/bartlein/Dropbox/WebPages/Bartlein/pjbartlein.github.io/source/BartleinAcadBio.pdf
/Users/bartlein/Dropbox/WebPages/Bartlein/pjbartlein.github.io/source/BartleinCV.pdf
/Users/bartlein/Dropbox/WebPages/Bartlein/pjbartlein.github.io/source/BartleinPublications.pdf

and
